## Overview

## Assignment Tasks
	1. Calculate the average film length for each category.  
	2. Identify the categories with the longest and shortest average film lengths.  
	3. List customers who rented Action movies but not Comedy or Classics.  
	4. Find the actor who appeared in the most English-language films.  
	5. Count distinct movies rented for exactly ten days from the store where Mike works.  
	6. List actors who appeared in the movie with the largest cast.  

	Each task includes the SQL query, query output, and a short analysis explaining how the problem was solved.
